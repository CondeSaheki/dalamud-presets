# dalamud-presets
Cli tool for enabling/disabling mods

DO NOT USE THIS WHILE DALAMUD IS RUNNING!

# Usage
like any cli app
```
dalamud-presets -help
```
response:
```
-help                         Show this help message
-show                         Show all mods and status
-enable                       Enable all mods
-disable                      Disable all mods
-p:display                    WIP     
-p:apply                      WIP     
-p:edit                       WIP     
```
